## üìä Weekly Study Schedule Template

### Weekdays (Mon, Wed, Thu, Fri)
- **1.5-2 hours**: Video lessons + coding along
- Focus: Theory and guided practice

### Tuesday (Work Day)
- **30 mins - 1 hour**: Light study or review
- Watch videos during lunch break if possible

### Weekends (Sat, Sun)
- **3-4 hours each day**: Project work
- Focus: Building and experimenting

### Study Tips
1. **Code along** - never just watch videos
2. **Build variations** - modify instructor's projects
3. **Take breaks** - Pomodoro technique (25 min work, 5 min break)
4. **Review regularly** - revisit concepts weekly
5. **Ask for help** - Stack Overflow, Discord communities
6. **Track progress** - check off completed topics
7. **Commit daily** - push code to GitHub every session
